  • Trip Duration (seconds)
  • Start Time and Date
  • End Time and Date
  • Start Station ID
  • Start Station Name
  • Start Station Latitude
  • start_station_longitude
  • bike_id
  • user_type
  • member_birth_year
  • bike_share_for_all_trip

  • Day_of_trip
